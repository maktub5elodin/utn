# Unidad VIII — Gestión Industrial

## Definición de Gestión

La **gestión** (management) en ingeniería es el conjunto de actividades de planificación, organización, dirección y control de los recursos (humanos, materiales, financieros, tecnológicos y de información) de una organización para alcanzar objetivos definidos.

El ingeniero gestor tiene mayor contacto con decisiones de negocios que con actividades directamente técnicas. No deja de ser ingeniero — su valor es comprender el sistema técnico y gestionarlo eficientemente.

Analogía contable: la gestión es para la fábrica lo que la gerencia financiera es para una empresa — traduce la realidad operativa en decisiones económicas y coordina todos los recursos.

## Funciones Clásicas de la Gestión (Fayol)

```
GESTIÓN
│
├── PLANIFICAR ──► definir objetivos y el camino para alcanzarlos
│
├── ORGANIZAR ──► asignar recursos y responsabilidades; diseñar la estructura
│
├── DIRIGIR ──► liderar al equipo; motivar; comunicar; resolver conflictos
│
├── CONTROLAR ──► comparar resultados vs. objetivos; detectar desvíos; corregir
│
└── COORDINAR ──► articular las distintas áreas para funcionar como sistema
```

## La Organización Industrial

La **organización industrial** define cómo se estructura una empresa para operar eficientemente:

### Estructura Funcional

```
GERENCIA GENERAL
      │
 ─────┴──────────────────────
 │          │         │
INGENIERÍA PRODUCCIÓN ADMINISTRACIÓN
 (diseño)  (fabricar)  (finanzas, RRHH)
```

Ventaja: especialización; personal altamente competente en su área.
Limitación: silos departamentales; baja coordinación entre áreas.

### Estructura por Proyecto (Matricial)

```
GERENCIA GENERAL
      │
 ─────┴──────────────────────
 │          │
Dir. PROYECTO   Dir. PRODUCCIÓN
      │              │
  Ing. Diseño ←──── asignado al proyecto
  Ing. Procesos ←── asignado al proyecto
```

El personal pertenece a un departamento funcional pero es asignado a proyectos. Permite flexibilidad.

## El Rol del Ingeniero Gestor

Un ingeniero recién graduado en la línea base aprende:
- Problemas reales de la operación
- Actividades documentadas y no documentadas
- El conocimiento práctico que tienen los operarios (a veces mayor que el del ingeniero junior)

Al ascender, este conocimiento real permite tomar decisiones informadas sobre lo que necesita la empresa. El ingeniero gestor combina conocimiento técnico con capacidad de liderazgo, comunicación y decisión económica.

## Gestión de Recursos Humanos

La gestión de personas en la industria incluye:

| Función | Descripción |
|---|---|
| Selección | Identificar y contratar el perfil adecuado |
| Capacitación | Formación técnica continua; actualización |
| Evaluación de desempeño | Medir resultados; retroalimentación |
| Motivación | Incentivos, reconocimiento, plan de carrera |
| Estructura salarial | Escala compatible con la dificultad del trabajo |
| Seguridad laboral | Condiciones de trabajo seguras y saludables |

El ingeniero mecánico que gestiona equipos debe entender que trabajadores de niveles operativos poseen conocimiento práctico profundo — saber escucharlos es clave para la mejora continua.

## Gestión de Operaciones

Planifica y controla el flujo de materiales, información y energía dentro de la planta:

```
PROVEEDOR ──► RECEPCIÓN ──► ALMACÉN MP ──► PRODUCCIÓN ──► ALMACÉN PT ──► CLIENTE
              (control          │                               │
              de calidad)   MRP/MPS                        control de
                                                            expedición
```

Indicadores de gestión de operaciones:
- **Lead time**: tiempo desde el pedido del cliente hasta la entrega
- **Nivel de servicio**: porcentaje de pedidos entregados en tiempo y forma
- **Rotación de inventario**: cuántas veces al año se renueva el inventario
- **Costo unitario de producción**: costo total / unidades producidas

## Gestión de Calidad

La gestión de calidad no es solo inspección al final — es un sistema que abarca todo el ciclo productivo:

```
CICLO PDCA (Plan - Do - Check - Act)

PLAN ──► DO ──► CHECK ──► ACT ──► (nuevo ciclo)
 │         │        │        │
Planificar  Ejecutar Medir  Corregir y
el cambio   el plan  resultados mejorar
```

**ISO 9001**: Norma internacional de sistemas de gestión de la calidad. Establece requisitos para que una organización demuestre que puede proporcionar consistentemente productos y servicios que satisfagan al cliente.

Analogía contable: ISO 9001 es como las normas NIIF — un marco de referencia que da confianza a los clientes sobre la calidad de los procesos, similar a cómo las NIIF dan confianza sobre la información financiera.

## Gestión Ambiental

El ingeniero gestor debe incorporar el impacto ambiental en todas las decisiones:

**ISO 14001**: Norma de sistemas de gestión ambiental. Objetivo: producir bienes y servicios minimizando el uso de recursos y sin ocasionar daño ambiental.

```
GESTIÓN AMBIENTAL (SGA)
├── Identificación de aspectos ambientales significativos
├── Objetivos y metas de reducción
├── Monitoreo y medición
└── Revisión y mejora continua
```

La ingeniería ambiental es parte integral de la formación del ingeniero mecánico — la preservación del medio ambiente no es opcional sino una responsabilidad profesional.

## Outsourcing (Tercerización)

Como consecuencia de los procesos de reingeniería empresarial, muchas empresas tercerizan actividades no esenciales:
- Mantenimiento de maquinaria
- Capacitación de personal
- Servicios de transporte y logística
- Tecnología de la información

La empresa conserva internamente solo las actividades que son **core** (fundamentales para su negocio).

El consultor que provee estos servicios debe tener: buen conocimiento técnico, capacidad financiera, liderazgo y habilidades de comunicación.

## Gestión de la Tecnología e Innovación

El ingeniero gestor también gestiona el **ciclo tecnológico** de la empresa:

```
TECNOLOGÍA
│
├── VIGILANCIA TECNOLÓGICA ──► monitorear el entorno; detectar nuevas tecnologías
├── EVALUACIÓN ──► analizar si la tecnología es aplicable y rentable
├── ADQUISICIÓN ──► comprar, licenciar o desarrollar la tecnología
├── ASIMILACIÓN ──► incorporar al proceso productivo; capacitar
└── INNOVACIÓN ──► mejoras propias; nuevos procesos o productos
```

## Resumen

| Función | Descripción |
|---|---|
| Planificar | Definir objetivos y estrategia |
| Organizar | Asignar recursos y estructura |
| Dirigir | Liderar y motivar al equipo |
| Controlar | Medir, detectar desvíos, corregir |
| PDCA | Ciclo de mejora continua |
| ISO 9001 | Gestión de calidad |
| ISO 14001 | Gestión ambiental |
| Outsourcing | Tercerización de actividades no esenciales |

## Errores conceptuales comunes

- Creer que gestión y administración son lo mismo: la administración es más amplia; la gestión es la aplicación práctica dentro de un contexto específico.
- Pensar que el control solo detecta errores: también confirma que el proceso funciona bien y genera datos para mejorar.
- Ignorar el capital humano: la tecnología es replicable; el conocimiento tácito del equipo no.
