# Unidad III — La Ingeniería como Actividad Profesional

## El Perfil del Ingeniero Mecánico

### Nivel 1 — Ingeniería estándar (la mayoría de los egresados)

Opera con tecnologías consolidadas, con metodologías de análisis y diseño probadas y documentadas en bibliografía amplia. Abarca:

- Proyecto mecánico
- Dirección de instalaciones y montajes industriales
- Diseño de productos industriales
- Administración de proyectos
- Organización industrial
- Programación del mantenimiento
- Pericias y asesoramientos técnicos
- Docencia técnica de grado

Requiere formación equilibrada: ciencias básicas + ciencias aplicadas + materias tecnológicas + economía y gestión + relaciones humanas.

### Nivel 2 — Ingeniería avanzada (especialistas y posgrado)

Opera en la frontera del conocimiento técnico. Enfrenta problemas de alta complejidad, precisión y escala. Roles típicos:

- Especialistas y consultores de mayor jerarquía
- Conductores de equipos de trabajo
- Investigación, desarrollo e innovación tecnológica
- Jefes de proyectos de ingeniería relevantes
- Profesores universitarios de posgrado

Requiere formación profunda en ciencias puras y gran capacidad de investigación.

## Las 10 Cualidades del Ingeniero

| N° | Cualidad | Descripción |
|---|---|---|
| 1 | Polifacético | Apto para varias disciplinas; ejecuta muchas actividades conectadas |
| 2 | Ingenioso | Busca soluciones tangibles; analítico y estratega |
| 3 | Creativo | Genera ideas y diseños a partir de la nada |
| 4 | Amor por la ciencia | Usa el conocimiento como punto de partida para crear, no solo para teorizar |
| 5 | Actitud analítica | Analiza hasta el mínimo detalle para proponer soluciones |
| 6 | Trabajador | Exigente mental y físicamente; activo y dinámico |
| 7 | Social | Trabaja en equipo; sabe integrar y comunicar |
| 8 | Mente lógica | Razonamiento lógico sobre pensamiento idealista |
| 9 | Detallista | Meticuloso; el mínimo error puede colapsar un sistema |
| 10 | Educación continua | La ingeniería avanza; el estudio no termina al graduarse |

## Competencias Profesionales

```
COMPETENCIAS
├─ Capacidad   = posibilidad de relacionar el conocimiento con la realidad
├─ Habilidad   = aplicación de la capacidad sobre una realidad específica
├─ Aptitud     = seleccionar entre capacidades y habilidades para una realidad compleja
└─ Actitud     = predisposición personal para la concreción de un objetivo
```

Todas las profesiones requieren estas cuatro competencias. De una profesión a otra varía el tipo de habilidades y capacidades requeridas.

Analogía contable: un contador tiene la *capacidad* de leer un balance, la *habilidad* de detectar inconsistencias, la *aptitud* de priorizar qué revisar primero y la *actitud* de hacerlo con rigor y honestidad. El ingeniero mecánico atraviesa el mismo esquema.

## Las 14 Funciones de la Ingeniería

| Función | Descripción |
|---|---|
| **Investigación básica** | Busca nuevos métodos y procedimientos para usar recursos naturales. Colabora con la ciencia. Los dept. de I+D priorizan la innovación. |
| **Desarrollo del producto** | Establece la aplicación práctica de un producto ya investigado. Trabaja en equipos interdisciplinarios. |
| **Diseño técnico** | Resuelve problemas específicos de diseño de partes del producto; elabora especificaciones, define módulos, distribuye el trabajo. |
| **Dirección del Depto. Técnico** | Dirige el personal, planifica el mantenimiento preventivo, resuelve problemas; requiere capacidad organizativa y relaciones humanas. |
| **Soporte técnico** | Analiza problemas de productos en campo y proporciona soluciones; incluye soporte preventivo (pruebas de estado). |
| **Proyectos** | Aplica metodología racional para encontrar soluciones detalladas a necesidades definidas inicialmente. |
| **Construcción** | Combina resultados previos para ensamblar o producir sistemas. Administra recursos económicos, materiales y mano de obra. |
| **Producción** | Similar a construcción pero repetida cientos o miles de veces. Requiere calidad constante. |
| **Operación** | Supervisa líneas de ensamble; busca rendimiento óptimo de equipos. Requiere sensibilidad para resolver imprevistos. |
| **Aplicación y ventas** | Vende productos tecnológicos complejos. Exige alto conocimiento técnico y grandes habilidades de comunicación. |
| **Industria** | Relacionada con el elemento humano: capacitación de personal, escalas salariales, rol de asesor. |
| **Administración** | Gestión de procesos/sistemas en puestos ejecutivos. Mayor contacto con decisiones de negocio que con tareas técnicas. |
| **Consultoría** | Outsourcing de actividades no centrales. El consultor debe conocer la disciplina, finanzas, y liderar equipos. |
| **Educación** | Formación continua y docencia. La tecnología avanza; los conocimientos se renuevan continuamente. |

Punto de entrada recomendado para el recién graduado: **producción** — pasar un período en la cadena base, conocer todas las actividades (documentadas y no documentadas), y entender los problemas reales de operación antes de ascender.

## Habilidades Básicas de la Ingeniería

### 1. Representación

```
REPRESENTACIÓN
├─ Icónica   = imágenes, planos, dibujos técnicos, esquemas
│              (lo que parece el objeto — formas, dimensiones)
└─ Diagramática = símbolos funcionales, diagramas de flujo, circuitos
                   (cómo funciona — relaciones y flujos)
```

Analogía: el plano de una bomba hidráulica (icónico) vs. el diagrama hidráulico del sistema (diagramático).

### 2. Optimización

Encontrar la mejor solución dentro de restricciones (costo, tiempo, materiales, seguridad). El ingeniero no busca la solución perfecta sino la **óptima dado el contexto**.

Analogía contable: el contador no busca el balance perfecto sino el que maximiza la información útil dentro de las normas vigentes.

### 3. Resolución de Problemas

La heurística (reglas de oro de la experiencia) permite acortar el camino hacia la solución, aunque no garantiza llegar. Sin experiencia el ingeniero debe seguir el camino de los libros. Con experiencia, encuentra atajos.

### 4. Intercambiabilidad

Capacidad de reemplazar una pieza por otra de las mismas especificaciones, sin ajuste manual. Fundamental para la producción en serie y el mantenimiento.

## Máquinas: Motores y Operatrices

```
MÁQUINAS
│
├─ MOTORES PRIMARIOS (generan potencia mecánica)
│   ├─ Máquinas de vapor        (Revolución Industrial I)
│   ├─ Motores de combustión interna  (gasolina, diesel, gas)
│   ├─ Turbinas de gas / vapor  (turbogeneradores, aviación)
│   ├─ Motores eléctricos       (corriente alterna / continua)
│   └─ Motores hidráulicos / neumáticos
│
└─ MÁQUINAS OPERATRICES (consumen potencia mecánica para hacer trabajo)
    ├─ Bombas               (trasladan fluidos)
    ├─ Compresores          (comprimen gases)
    ├─ Ventiladores         (mueven aire)
    ├─ Generadores eléctricos   (convierten mecánica → electricidad)
    └─ Máquinas herramienta     (tornos, fresadoras, rectificadoras)
```

La potencia generada por el motor primario se transmite a la máquina operatriz mediante ejes, correas, engranajes o acoplamientos hidráulicos.

## Mercado Laboral — Argentina

- Argentina: 1 ingeniero cada 6.000 habitantes
- Brasil: 1 cada 4.000
- Alemania: 1 cada 2.000
- 7 de cada 10 industrias tiene dificultad para reclutar ingenieros (Centro Argentino de Ingenieros)
- Sectores con mayor demanda: automotriz, siderurgia, petróleo

El ingeniero mecánico en la industria petrolera tiene rol crítico: el cuidado de equipos y el mantenimiento de plantas que operan 365 días al año. Una falla puede generar desabastecimiento y consecuencias ambientales.

## Errores conceptuales comunes

- Creer que la función del ingeniero es solo diseñar: hay 14 funciones; la mayoría no son diseño puro.
- Confundir técnico e ingeniero: el técnico sabe *cómo*, el ingeniero sabe *por qué*.
- Pensar que la heurística reemplaza al conocimiento: acelera el camino, pero sin base teórica no puede aplicarse.
