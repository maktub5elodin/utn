# Unidad IX — Seguridad e Higiene Industrial

## Definición y Marco Conceptual

La **seguridad e higiene industrial** es la disciplina que identifica, evalúa y controla los riesgos en el ambiente de trabajo para proteger la salud y la integridad física de los trabajadores, las instalaciones y el medio ambiente.

No es una actividad separada de la ingeniería — es una dimensión **transversal a todos los procesos**. Un diseño de maquinaria inseguro, una instalación que no cumple normas o un procedimiento operativo deficiente pueden causar accidentes graves.

Analogía contable: la seguridad es como el control interno contable — no evita que se haga el trabajo, pero establece las salvaguardas para que se haga sin errores que causen daño irreversible.

## Conceptos Básicos

### Riesgo vs. Peligro

```
PELIGRO (Hazard)
└── fuente, situación o acto con potencial de causar daño
    Ej: un depósito de gas a presión

RIESGO (Risk)
└── combinación de la probabilidad de que ocurra el daño
    y la severidad del mismo
    Ej: la probabilidad de que explote el depósito × la consecuencia
```

```
Riesgo = Probabilidad × Consecuencia
```

### Accidente vs. Incidente

| Concepto | Definición |
|---|---|
| Accidente | Evento no deseado que produce lesión, daño o pérdida |
| Incidente | Evento no deseado que no produjo daño pero tenía potencial de hacerlo |
| Casi-accidente (Near-miss) | Incidente muy cercano a convertirse en accidente |

Gestionar incidentes y casi-accidentes previene accidentes graves — cada incidente es una señal de alerta del sistema.

## Marco Legal en Argentina

La principal normativa argentina en seguridad e higiene industrial:

| Norma | Contenido |
|---|---|
| **Ley 19.587** (1972) | Ley de Higiene y Seguridad en el Trabajo. Marco general. |
| **Decreto 351/79** | Reglamentación de la Ley 19.587. Detalla requisitos técnicos. |
| **Ley 24.557** (1995) | Ley de Riesgos del Trabajo (LRT). Sistema de ART. |
| **Ley 24.449** | Ley de Tránsito (aplicable a vehículos industriales) |
| Normas IRAM | Normas técnicas nacionales (complementan la legislación) |

Las **ART (Aseguradoras de Riesgos del Trabajo)** son las entidades privadas que cubren económicamente los accidentes y enfermedades laborales.

## Identificación de Riesgos — Tipos

```
RIESGOS EN EL AMBIENTE INDUSTRIAL
│
├── FÍSICOS
│   ├─ Ruido (hipoacusia)
│   ├─ Temperatura extrema (calor / frío)
│   ├─ Radiaciones (UV, ionizantes)
│   ├─ Vibraciones
│   └─ Iluminación inadecuada
│
├── MECÁNICOS (accidentes)
│   ├─ Atrapamiento por máquinas en movimiento
│   ├─ Golpes por caída de objetos
│   ├─ Cortes y proyecciones de partículas
│   └─ Caídas (mismo nivel / distinto nivel)
│
├── QUÍMICOS
│   ├─ Exposición a vapores y gases tóxicos
│   ├─ Contacto con sustancias corrosivas
│   └─ Incendio / explosión (materiales inflamables)
│
├── ELÉCTRICOS
│   ├─ Contacto eléctrico directo
│   └─ Contacto eléctrico indirecto (falla de aislación)
│
└── ERGONÓMICOS
    ├─ Posturas inadecuadas
    ├─ Levantamiento manual de cargas
    └─ Movimientos repetitivos
```

## Evaluación y Control de Riesgos

### Jerarquía de Controles (más efectivo → menos efectivo)

```
1. ELIMINACIÓN ──────────► Eliminar el peligro por completo
2. SUSTITUCIÓN ──────────► Reemplazar por algo menos peligroso
3. CONTROLES DE INGENIERÍA ► Aislar o reducir el peligro (guardas, ventilación)
4. CONTROLES ADMINISTRATIVOS ► Procedimientos, capacitación, señalización
5. EPP (Equipos de Protección) ► Casco, guantes, antiparras, protección auditiva
```

Los controles de ingeniería (nivel 3) son preferibles a los administrativos y al EPP porque no dependen del comportamiento humano.

### Matriz de Riesgo

```
              CONSECUENCIA
               LEVE  MODERADA  GRAVE  CATASTRÓFICA
PROBABILIDAD ─────────────────────────────────────
Muy probable  │ MED │  ALTO  │ CRIT │    CRIT     │
Probable      │ BAJ │  MED   │ ALTO │    CRIT     │
Posible       │ BAJ │  BAJ   │ MED  │    ALTO     │
Improbable    │ AEP │  AEP   │ BAJ  │    MED      │

AEP = Aceptable  BAJ = Bajo  MED = Medio  ALTO = Alto  CRIT = Crítico
```

## Dispositivos de Seguridad en Maquinaria

```
PROTECCIONES DE MÁQUINAS
├── Guardas fijas: barreras físicas permanentes (rejillas, tapas)
├── Guardas móviles: se abren para acceso; bloquean al cerrarse
├── Dispositivos de enclavamiento: detienen la máquina si se abre la guarda
├── Barras de parada de emergencia: el operario activa la parada con el cuerpo
├── Cortinas de luz: detectan la presencia de manos/cuerpos
└── Botón de emergencia (paro de emergencia): rojo, fácilmente accesible
```

## Equipo de Protección Personal (EPP)

El EPP es la **última línea de defensa**. No elimina el riesgo; reduce la severidad de las lesiones si el riesgo se materializa.

| Zona | EPP |
|---|---|
| Cabeza | Casco de seguridad |
| Ojos / cara | Antiparras, pantalla facial |
| Oídos | Tapones, orejeras |
| Vías respiratorias | Barbijo, respirador con filtro |
| Manos | Guantes (anticorte, antivibratorio, dieléctrico) |
| Pies | Calzado de seguridad con puntera de acero |
| Cuerpo | Ropa ignífuga, chaleco reflectivo |
| Altura | Arnés de seguridad |

## Incendio y Explosión

Los incendios requieren tres elementos simultáneos:

```
TRIÁNGULO DEL FUEGO

        COMBUSTIBLE
         /       \
        /         \
      CALOR ── OXÍGENO

Eliminar cualquiera de los tres ──► extingue el fuego
```

Clases de fuego y agentes extintores:

| Clase | Material | Agente extintor |
|---|---|---|
| A | Sólidos (madera, papel, textiles) | Agua, polvo seco ABC |
| B | Líquidos y gases inflamables | CO₂, polvo seco, espuma |
| C | Equipos eléctricos energizados | CO₂, polvo seco (nunca agua) |
| D | Metales combustibles (Mg, Na) | Polvo especial |

## Señalización Industrial

Colores y formas normalizados (ISO 3864, IRAM):

```
ROJO     → Prohibición / Peligro / Equipo contra incendios
AMARILLO → Precaución / Advertencia de riesgo
VERDE    → Seguridad / Salida de emergencia / Primeros auxilios
AZUL     → Obligación (usar EPP, no usar sin autorización)
```

## Higiene Industrial — Enfermedades Profesionales

La higiene industrial identifica y controla agentes ambientales que pueden causar enfermedades en los trabajadores:

| Agente | Enfermedad asociada |
|---|---|
| Ruido > 85 dB (8h) | Hipoacusia (pérdida auditiva) |
| Polvo de sílice | Silicosis (enfermedad pulmonar) |
| Asbesto | Asbestosis, mesotelioma |
| Plomo | Saturnismo (intoxicación por plomo) |
| Vibración (herramientas) | Síndrome de Raynaud |
| Temperatura extrema | Golpe de calor / hipotermia |

## Seguridad en Sistemas de Alta Presión

La ingeniería mecánica trabaja frecuentemente con sistemas de presión (tuberías, recipientes, calderas). Riesgos:
- Explosión por sobrepresión
- Fuga de fluidos tóxicos o inflamables
- Fatiga de materiales por ciclos de presión

Normas aplicables: ASME Boiler & Pressure Vessel Code; normas IRAM para recipientes a presión.

El diseño de alta presión requiere:
- Materiales resistentes a la temperatura y corrosión
- Válvulas de seguridad calibradas
- Inspecciones periódicas certificadas

## Resumen

| Concepto | Descripción |
|---|---|
| Peligro | Fuente con potencial de causar daño |
| Riesgo | Probabilidad × Consecuencia |
| Accidente | Evento con daño real |
| Incidente | Evento sin daño, con potencial |
| Jerarquía de controles | Eliminación > sustitución > ingeniería > administrativo > EPP |
| Triángulo del fuego | Combustible + calor + oxígeno |
| Ley 19.587 | Marco legal nacional de seguridad e higiene |

## Errores conceptuales comunes

- Confundir peligro con riesgo: el depósito de gas es un peligro; el riesgo depende de la probabilidad y consecuencia.
- Creer que el EPP es suficiente: es la última opción; los controles de ingeniería son más efectivos.
- Pensar que la seguridad es solo reglamentaria: un sistema seguro reduce costos, ausentismo, accidentes y mejora la productividad.
